# House Price Prediction Multivariate Linear Regression Model
A Python script that uses multivariate linear regression to predict house prices based on the area, number of bedrooms,
and distance from the city center.

# Files
1. "multivariate_linear_regression_model.py" - the main Python script.
2. "housing_data.xlsx" - dataset with 10 records.

# How to Run?
1. Clone this repository.
2. Install the requirements. (see the file attached "requirements.txt")
3. Run: python multivariate_linear_regression_model.py
